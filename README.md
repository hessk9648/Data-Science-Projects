# Data-Science-Projects

Portfolio repository for data science coursework and independent projects.

## Portfolio Site
- Main page: `index.html`
- Sections included: About, Work/Projects, Contact
- Repository link is available from the site navigation and contact section

## Project Structure
Projects are organized under `/projects`, with each project folder containing:
- the notebook (`.ipynb`)
- a project `README.md` summary

1. [ALS Indicators](projects/als-indicators/README.md) - Clustering analysis of anonymized ALS patient data.

## Setup
1. Create and activate a Python virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open notebooks in JupyterLab or Jupyter Notebook.

## Notes
- This repository currently has no automated lint/build/test pipeline configured.
- Notebook execution may require local data files referenced within individual notebooks.
